# Analyse de données de biocapteurs

## Type de projet
Projet académique en sciences du vivant.

## Contexte
Analyse de signaux biologiques issus de biocapteurs dans une logique
capteur → signal → traitement → interprétation.

## Objectif
Explorer et interpréter des données biologiques afin de comprendre
comment des signaux peuvent être exploités pour interagir avec des
systèmes ou des machines.

## Méthodes
- Analyse de données
- Visualisation de signaux biologiques
- Interprétation quantitative

## Outils
- Python
- Jupyter Notebook

## Lien avec la neuro-ingénierie
Ce projet s’inscrit dans une démarche de compréhension et
d’exploitation de signaux biologiques, étape clé dans la conception
d’interfaces entre le vivant et les machines.
